# 開発ログ

このファイルには、金融数学研究コックピットの開発の経緯を時系列で記録します。
新しい変更を加えたら、このファイルの一番下に追記していきます。

---

## 2026-07-20

### 初期構築
- プロジェクト一式（`index.html` / `style.css` / `app.js` / `supabase/schema.sql`）を作成。
- Supabaseプロジェクトを新規作成し、テーブル（`tasks` / `formulas` / `research_logs` / `shared_draft`）とRealtime・RLSを設定。
- GitHubリポジトリを作成し、GitHub Pagesで公開（`https://k-kei1412.github.io/financial-math-cockpit/`）。

### レイアウト修正
- チャットエリア・タスクエリアが画面を圧迫し、公式ライブラリや研究ログ履歴が見えなくなっていた問題を修正。ページ全体がスクロールできるようにし、中央エリアに最低限の高さを確保。

### 公式ライブラリの改善
- 公式カードをアコーディオン形式に変更（クリックで開閉、複数同時に開ける）。
- 「新しい公式を登録」フォームを検索窓の上に移動。
- 登録時に似た名前の公式がないか警告表示する機能を追加。

### 複数人利用まわりの機能追加
- 接続中のメンバー一覧をヘッダーの「接続者」から見られるように（Supabase Realtime Presenceを使用。同じ名前なら同一人物として扱う）。
- 入室ゲート（名前を入力するまでアプリを使えない画面）を追加。

### 管理者機能（第1弾）
- `admin.html` にパスワード（`23042`）でログインすると、チャットメッセージを個別削除できる管理者モードを追加。

### 削除まわりの見直し（バックアップ対策）
- 全ての削除（タスク・研究ログ・チャット・公式）を論理削除（`deleted_at`）に変更。誤って消してもゴミ箱から復元可能に。
- タスク・研究ログの削除ボタンは管理者のみ表示に変更。
- 公式の編集・削除機能を管理者向けに追加。
- チャットの一括削除ボタン（管理者のみ）を追加。
- ゴミ箱に「全て復元」ボタンを追加。
- スマホ・タブレット向けのレスポンシブレイアウト（画面が狭い時は3カラムを縦積みに）を追加。
- サイトアイコン（ノートとクラウド同期のイラスト）を favicon として設定。

### 管理者ログイン方式の変更
- `admin.html` によるパスワードページ方式を廃止。
- 代わりに、表示名として `23042` と入力すると自動的に管理者になる方式に変更。管理者になると表示名は常に「管理者」と表示され（`23042`という文字は誰にも見えない）、名前を変更しない限り管理者権限は外れない。
